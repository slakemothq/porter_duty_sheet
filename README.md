# Night Porter Duty Sheet

Night Porter duty tracking app for Officers' Mess, CTCRM.

## Deploy to Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your GitHub repo: `SlakemothQ/porter-duty-sheet`
4. Render auto-detects the `render.yaml` config
5. Click **Deploy** — done

## Local development

```bash
npm install
npm start
```

App runs at `http://localhost:3000`
